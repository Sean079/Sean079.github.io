Hangman Game

Home page & high score table

For this project, we were given the task of utilizing Websockets for asynchronous communication to create a Hangman game utilizing a login, logout, and register page, whereby a user's username and hashed-and-salted password would be stored in a database. When playing a game, whenever the player successfully guesses their word, it will store it into a high score table sorted based on both their word length and how many tries they have left. You can access the code repo here.
