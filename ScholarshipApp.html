Scholarship Application

For this project, we were given the task to upgrade an existing scholarship application created by a previous capstone class from .NET 2.0 to .NET 6.0, as well as find and fix a variety of bugs associated with the application. These varied from creating an administrator on first time setup to a crash occurring when an administrator was trying to look at awarded scholarships when nothing was yet rewarded.

Technologies

  • Razor Pages
  • .NET Core
  • Entity Framework
  • HTML5
