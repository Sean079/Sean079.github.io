California Speed Card Game

Start of game

For this project, we were given the task of creating a realtime two player card game California Speed. Utilizing SignalR to handle communication between the players, JavaScript was used to handle the card interactions, with C# code in the background to handle the game logic for comparing the cards and sending new lists or stacks the front end utilizes to update the visuals. You can access the code repo here.

Technologies
