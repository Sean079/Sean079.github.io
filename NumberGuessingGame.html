Number Guessing Game

Register page

For this project, we were given the task of individually creating a number guessing game, where the user guess a number between 1 and 100 against a randomly generated number. Utilizing epizy.com for both the website and the database, the user login info, when registered, needed to be salted and hashed for security, and a high score table to display the top 10 user scores. You can access the code repo here.
